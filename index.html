Summary: Test Data

The purpose of this project was to analyze the district-wide standardized test results. To accomplish this, I accessed student, budget, test score, and school data from the district database.

When analyzing this data, I used matplotlib to create visualizations of multiple aspects that influence students' test results. Some of these factors included

  • math score by grade,
  • reading score by grade,
  • scores by school spending,
  • scores by school size, and
  • scores by school type.
